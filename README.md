# financial-market-model-matlab
### Heuristics and Systems Theory (University Project)

#### An end-to-end project about modelling financial market behaviour using a non-linear dynamic system. The model captures interactions between price, investor sentiment, and trading activity. Jacobian Linearization and Eigenvalue Analysis were used to evaluate system stability, while State-Feedback Control was applied to stabilise unstable dynamics. A Luenberger Observer was also implemented to estimate hidden states, and simulations were conducted to validate system performance.

### Tools: MATLAB
### Analysis: Non-Linear Modelling, Jacobian Linearization, Eigenvalue Analysis, State-Feedback Control, Luenberger Observer
